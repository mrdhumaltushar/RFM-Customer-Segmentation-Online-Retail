# RFM Customer Segmentation – Online Retail  

**Goal**: Segment 5,860 customers using **Recency, Frequency, Monetary (RFM)** to drive targeted marketing.

---

## Key Results
| Metric               | Value                     |
|----------------------|---------------------------|
| **Customers**        | **5,860**                 |
| **Total Revenue**    | **£17.3M**                |
| **Top 5 Customers**  | **£2.0M+ (12%)**          |
| **Champions**        | **66% of revenue**        |
